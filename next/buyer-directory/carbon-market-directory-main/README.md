# Carbon Market Directory

## Project Overview

The Carbon Market Directory is a comprehensive platform designed to provide insights into carbon credit buyers, market trends, and industry news. The application serves as a central hub for carbon market professionals, offering detailed information on companies purchasing voluntary carbon credits, their preferences, and market activities.

### Key Features

- **Buyer Directory**: Searchable database of carbon credit buyers with filtering capabilities
- **News Aggregator**: Real-time RSS feed integration from leading carbon market news sources
- **Analytics Dashboard**: Visual representation of market trends and buyer statistics
- **Responsive Design**: Fully responsive interface that works across desktop and mobile devices

## Architecture

The application is built using Next.js 14 with the App Router, leveraging React Server Components for improved performance and SEO. It uses Supabase for database functionality and integrates with external RSS feeds for news content.

### Tech Stack

- **Frontend**: Next.js 14, React, Tailwind CSS, shadcn/ui components
- **Backend**: Next.js API routes, Server Actions
- **Database**: Supabase (PostgreSQL)
- **State Management**: React Hooks
- **Styling**: Tailwind CSS with custom British-inspired theme
- **Data Visualization**: Recharts
- **External Integrations**: RSS feeds via rss-to-json
