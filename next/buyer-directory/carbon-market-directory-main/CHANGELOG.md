# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2024-03-19

### Added
- Initial release of the Carbon Market Directory
- Buyer directory with search and filtering capabilities
- News aggregator with RSS feed integration
- Analytics dashboard with market insights
- Responsive design for all screen sizes
- Supabase integration for database functionality
- Comprehensive documentation

### Fixed
- RSS feed integration to properly handle errors and fallbacks
- Corrected import of rss-to-json library
- Added robust error handling for API routes

## [0.9.0] - 2024-03-15

### Added
- Pagination for the directory table
- Server-side filtering for the directory
- Analytics charts and visualizations
- News feed component with RSS integration

### Changed
- Updated Supabase table name to buyers_public
- Improved error handling and loading states
- Enhanced responsive design for mobile devices

## [0.8.0] - 2024-03-10

### Added
- Initial directory implementation
- Basic filtering and search functionality
- Supabase integration for database access
- UI components and styling
